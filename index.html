About Obsidian

Obsidian is a privacy-first WhatsApp chat viewer. It transforms exported .txt and .zip files into a beautiful, searchable, and responsive interface.

Capabilities

  • Client-side Parsing: Reads .txt and .zip exports instantly.
  • Search: Real-time message search with highlighting.
  • Analytics: View message counts and top senders.
  • Theming: Toggle between Dark and Light modes.

Limitations

  • Media: Images and videos cannot be displayed directly from the text file. They are marked as "Media omitted".
  • Voice Notes: Audio playback is not currently supported.

How it Works

This tool uses the File Reader API to process your chat export entirely within your browser memory. No data is ever uploaded to a server.
